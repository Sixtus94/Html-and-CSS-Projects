Using a media query in CSS, set the font color of the paragraph element to blue if the screen is smaller than 600px

Use mediaqueries to set the background-color to red if the viewport is 600 pixels wide or wider, to lightgreen if the viewport is between 400 and 599 pixels wide. If the viewport is smaller than 400 pixels, the background-color is lightgreen.
